# hello-world-node API

API básica em Node

## Ambientes

| Env | URL                         |
| --- | --------------------------- |
| LOCAL | http://localhost:5000/health |
| PROD  | https://node-server-test-5svf.onrender.com | 

### Como executar

Instalando as dependências:

```
$ npm i
```
Roda a aplicação em modo desenvolvimento

```
$ npm run start:dev
```

### Como testar

```
$ npm run test
```
